Voraussetzungen (APO-SI)

MSA-Q (Qualifikation)

  • Mindestens 3 E-Kurse auf dem Zeugnis.
  • Hauptfächer: Mindestens Note 3 (im G-Kurs Note 2).
  • Nebenfächer: Mindestens Note 3.
  • Wichtig: Keine Note 5 oder 6 erlaubt.
  • Joker: Einmal Note 4 im Nebenfach ist frei.

MSA (Realschule)

  • Mindestens 2 E-Kurse auf dem Zeugnis.
  • Hauptfächer: Mindestens Note 4 (im G-Kurs Note 3).
  • Nebenfächer: Zweimal Note 3 nötig.
  • Wichtig: Keine Note 6 erlaubt.

EESA (HA 10)

  • Hauptfächer: Deutsch, Mathe, Arbeitslehre, Naturwissenschaften.
  • E-Kurs Bonus: Note 5 zählt wie im G-Kurs Note 4.
  • Max. eine Note 6 (Ausgleich durch Note 2 zwingend).
  • Insgesamt maximal 2 Defizite erlaubt.
