Voraussetzungen (APO-SI)

Wichtige Grundregel

Maximal ein Hauptfach und ein Nebenfach dürfen ausgeglichen werden.
Jede Ausgleichsnote gilt nur einmal.

MSA-Q (Qualifikation)

  • Mindestens 3 E-Kurse.
  • Hauptfächer: Note 3, im G-Kurs Note 2.
  • Nebenfächer: Note 3 (im G-Kurs Note 2).
  • Maximal 1 Defizit pro Fächergruppe. Ausgleich zwingend durch Note 2.
  • Keine 5 in Hauptfächern erlaubt.
  • Keine 6 erlaubt.

MSA (Realschule)

  • Mindestens 2 E-Kurse.
  • Hauptfächer: Note 4, im G-Kurs Note 3.
  • Nebenfächer: Note 4 (im G-Kurs Note 3), aber mindestens zweimal Note 3.
  • Maximal 1 Defizit pro Fächergruppe. Ausgleich zwingend erforderlich.
  • Keine 6 erlaubt.

EESA (HA 10)

  • Alle Fächer: Mindestens Note 4.
  • Eine Note 5 im E-Kurs gilt als Note 4.
  • Maximal 2 Defizite insgesamt, höchstens 1 davon im Hauptfach. Ausgleich zwingend.
  • Maximal eine 6 erlaubt. Ausgleich zwingend durch Note 2.

ESA (HA 9) & Abgang

  • Alle Fächer: Mindestens Note 4.
  • Maximal 2 Defizite insgesamt, höchstens 1 davon im Hauptfach. Ausgleich zwingend.
  • Generell keine 6 erlaubt.
  • Wer in Klasse 10 die EESA-Ziele verfehlt, geht mit dem ESA aus Klasse 9 ab.
